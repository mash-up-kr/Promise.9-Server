# [정책] 폴더

## 1. 폴더 생성
- 사용자당 최대 30개의 폴더를 생성할 수 있다.
- 폴더명은 최대 20자까지 입력할 수 있다.
- 동일한 폴더명 생성이 가능하다.

## 2. 폴더 수정
- 사용자는 생성한 폴더의 이름을 수정할 수 있다.
- 전체, 미분류, 최근 삭제된 항목 폴더의 이름은 수정할 수 없다.

## 3. 폴더 삭제 가능 여부
- 사용자는 생성한 폴더를 삭제할 수 있다.
- 폴더 삭제 시 포함된 링크도 최근 삭제된 항목으로 이동된다.
- 전체, 미분류, 최근 삭제된 항목 폴더는 삭제할 수 없다.

## 4. 기본 폴더 규칙
- 전체 폴더는 모든 링크를 조회할 수 있다.
- 미분류 폴더는 폴더가 지정되지 않은 링크를 조회할 수 있다.
- 최근 삭제된 항목 폴더는 삭제된 링크를 조회할 수 있다.
- 기본 폴더(전체, 미분류, 최근 삭제된 항목)는 생성, 수정, 삭제할 수 없다.
