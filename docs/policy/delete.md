# [정책] 삭제

## 1. 최근 삭제된 항목으로 이동
- 삭제된 링크는 최근 삭제된 항목 폴더로 이동되며 즉시 영구 삭제되지 않는다.
- 폴더 삭제 시 폴더에 포함된 링크도 함께 최근 삭제된 항목 폴더로 이동한다.
- 최근 삭제된 항목의 링크는 검색 결과에 포함하지 않는다.

## 2. 30일 임시 보관
- 삭제된 링크는 최근 삭제된 항목 폴더에 30일간 보관된다.
- 보관 기간 내 복원이 가능하다.

## 3. 자동 영구 삭제
- 삭제 후 30일이 경과한 링크는 자동으로 영구 삭제된다.
- 영구 삭제된 링크는 복구할 수 없다.

## 4. 복원 기능
- 사용자는 최근 삭제된 항목에서 링크를 복원할 수 있다.
- 복원된 링크는 미분류 폴더로 이동한다.
