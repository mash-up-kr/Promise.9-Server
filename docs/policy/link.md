# [정책] 링크 저장


## 1. URL 유효성 검사
- 저장 전 URL 형식 유효성 검사를 수행한다.
## 2. 중복 URL 정책
- 동일 URL은 중복 저장할 수 없다.
## 3. 폴더 지정 정책
- 저장 시 폴더는 최대 1개 선택할 수 있다.
- 폴더를 선택하지 않은 경우 미분류 폴더에 자동 저장된다.
- 저장 후 폴더 변경이 가능하다.
## 4. 메타 데이터 수집
- 저장 시 제목, 출처, 대표 이미지를 자동 수집한다.
- 수집 실패 시 URL만 저장한다.
- 대표 이미지가 없는 경우 기본 이미지를 노출한다.
## 5. 메모 정책
- 메모는 선택 입력 항목이다.
- 최대 500자까지 입력할 수 있다.
## 6. 저장 기본값
- 메모 기본값 : null
- 폴더 선택 기본값 : "미분류"
## 7. 저장 일시
- 링크 저장 시 저장 일시를 기록한다.
