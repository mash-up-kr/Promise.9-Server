## 문서 추가 규칙

- 새 문서는 목적이 드러나는 폴더 아래에 둡니다.
- `docs/README.md`에는 폴더 단위 링크만 둡니다.
- 세부 문서 링크는 각 폴더의 `README.md`에 둡니다.
- 한 문서에 여러 주제를 섞지 않습니다.
- 팀 컨벤션과 실제 레포 설정이 다르면 문서에 그 차이를 명시합니다.
